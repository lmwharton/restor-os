## Overview

**Problem:** Equipment is tracked informally (count x days for billing). No inventory management, no ROI tracking. Drying certificates are generated manually or not at all — but they're required to close out mitigation billing.

**Solution:** Equipment library with deployment tracking + auto-generated drying certificate from existing moisture readings data.

**Scope:**
- IN: Equipment library (CRUD, serial numbers, deployment tracking, ROI), drying certificate generation (PDF, room-by-room summary, compliance statement)
- OUT: Equipment alerts on dashboard (those fire from 04-I when this data exists — additive, not blocking)

## Technical Approach

**Equipment:** CRUD with deployment tracking. Deploy/remove creates timestamped records linking equipment to job/room. ROI is computed from deployment duration x billing rate.

**Drying Certificate:** Reads from existing `moisture_readings` and `job_rooms` tables. PDF generation via same pattern as job report (Spec 01). Determines completion by comparing final readings against dry standards.

**Key Files:**
- `backend/api/equipment/` — new: equipment library endpoints
- `backend/api/reports/drying_certificate.py` — new: certificate generation
- `web/src/app/(protected)/settings/inventory/` — new: equipment library UI
- `web/src/app/(protected)/jobs/[id]/` — add equipment section + certificate button

## Decisions & Notes

- **Renamed from 04C.** Same content, new numbering for consolidated 04-series.
- **Equipment billing is count x days, not serial numbers.** But inventory management helps owners track what they own and where it is.
- **Drying Certificate format** from Brett's ScopeFlow demo. Required to close out mitigation billing — proves equipment days were justified.
- **Equipment ROI tracking** is a value-add: which units generate most billing, flag underperformers.
- **Equipment alerts (dashboard):** When 04-I is built, equipment data can trigger alerts (malfunction, delivery needed). This is additive — 04-III doesn't need 04-I, and 04-I doesn't need 04-III. They enhance each other when both exist.
